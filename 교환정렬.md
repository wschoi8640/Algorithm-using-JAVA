# 교환정렬

* 문제 : 비내림차순(non-decreasing order)으로 n 개의 키를 정렬하라.

* 입력 : 양의 정수 n, 키의 배열 S(첨자는 1부터 n까지)

* 출력 : 키가 비내림차순으로 정렬된 배열 S

```java
// 순차적으로 비교하며 작은 수를 앞으로 당긴다
void exchangeSort (int n, int [] S)
{
	int i, j;
	
	for (i = 1; i <= n - 1; i++)
	{
		for (j = i + 1; j <= n; j++)
		{
			if (S[j] < S[i])
			{
				// 작은 수를 앞으로 당긴다
				exchange(i,j);
			}
		}
	}
}
```
